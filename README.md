## Project Description

This Android application calculates petrol costs and BUDI MADANI rebates based on user inputs. The application allows users to select a petrol type, enter petrol price and fuel usage, and determine eligibility for the BUDI MADANI rebate.

## Features

* Select petrol type (RON95, RON97, Diesel)
* Calculate total petrol cost
* Calculate BUDI MADANI rebate
* Display total savings
* Input validation
* About page with student information
* Menu navigation (Home and About)

## Formula Used

* Total Petrol Cost = Fuel Usage × Petrol Price
* BUDI Rebate = Fuel Usage × RM1.99
* Total Saving = Total Petrol Cost − BUDI Rebate
